RecipeApp

The Recipe App is a user-friendly web application that allows users to search for and explore their favorite recipes. The app fetches recipe data from The Meal DB API, offering a seamless culinary experience. Key features include a search box for recipe queries, a grid of recipe cards displaying basic information, and the ability to view detailed information about selected recipes. The app is designed with a responsive layout, ensuring compatibility across various screen sizes. Styled with CSS and utilizing Font Awesome icons, the project's file structure includes main files like index.html, style.css, and script.js for the app's structure, styling, and logic, respectively. Technologies used in this project include HTML5, CSS3, and JavaScript (ES6+), with Font Awesome providing a collection of icons. The Meal DB API is integrated to source recipe data for the application.

Mancala Game

The Mancala Game project introduces an engaging graphical interface (GUI) for the classic Mancala game, featuring two rule sets: Kalah Rules and Ayoayo Rules. In the Kalah version, players strategically distribute stones, depositing them in stores and capturing opponent's pieces to secure victory. Ayoayo follows a similar initiation, with the unique feature of capturing opponent's stones when the last stone lands in an empty pit on the player's side. The player with the most stones in their store at the conclusion of each version emerges as the triumphant strategist. The game requires the Java Development Kit (JDK) and Gradle for building, and the program's execution involves compiling files and running the TextUI class. Acknowledgments go to traditional Mancala Rules, online Mancala Games, and insightful resources like "awesome-readme" and "simple-readme" that contributed to the project's development.
