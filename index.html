Mancala Game - Java, JUnit, Event-driven programming

The Mancala Game project introduces an engaging JavaFX-based graphical interface (GUI) for the classic Mancala game, featuring two rule sets: Kalah Rules and Ayoayo Rules. Developed using the Java Development Kit (JDK) and built with Gradle, the project incorporates JUnit for unit testing and follows an MVC architecture to separate concerns effectively. The program's execution involves compiling files and running the TextUI class. Acknowledgments go to traditional Mancala Rules, online Mancala Games, and insightful resources like "awesome-readme" and "simple-readme" that contributed to the project's development.
